# GitHub Copilot Instructions для проекта DezTehUg

## Описание проекта
Это веб-сайт компании DezTehUg - услуги дезинфекции, дератизации и дезинсекции.

## Технологический стек
- **Framework**: Next.js 13.5.1 с App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI + shadcn/ui
- **Animations**: Framer Motion
- **Icons**: Lucide React + Custom SVG
- **State Management**: Zustand
- **Forms**: React Hook Form + Zod

## Структура проекта
```
app/                    # Next.js App Router pages
components/            # React компоненты
  ui/                 # shadcn/ui компоненты
  layout/             # Layout компоненты
  cyberpunk/          # Cyberpunk themed компоненты
public/               # Статические файлы
  icons/              # SVG иконки
lib/                  # Утилиты
hooks/                # Custom hooks
context/              # React Context
```

## Стиль кодирования
- Используй функциональные компоненты с TypeScript
- Предпочитай named exports
- Используй Tailwind CSS для стилизации
- Следуй принципам компонентной архитектуры
- Используй TypeScript интерфейсы для типизации props

## Naming Conventions
- Компоненты: PascalCase (например, `ServiceIcon`)
- Файлы компонентов: PascalCase.tsx
- Хуки: camelCase с префиксом `use`
- Константы: UPPER_SNAKE_CASE
- Переменные и функции: camelCase

## Предпочтительные паттерны
- Используй `interface` вместо `type` для объектов
- Предпочитай композицию наследованию
- Используй React.FC только когда необходимо
- Деструктуризация props в параметрах функции
- Используй optional chaining (?.) и nullish coalescing (??)

## Специфика проекта
- Тема: Cyberpunk/технологичный дизайн
- Цветовая схема: Неоновые цвета, темный фон
- Анимации: Плавные переходы с Framer Motion
- Иконки: Кастомные SVG + Lucide React
- Адаптивность: Mobile-first подход

## Компоненты для переиспользования
- `CyberpunkButton` - стилизованные кнопки
- `CyberpunkCard` - карточки в cyberpunk стиле
- `ServiceIcon` - иконки услуг
- `TrustFactorIcon` - иконки факторов доверия
- `WhyChooseUsIcon` - иконки преимуществ
