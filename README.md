# DezTechYug Website

A Next.js project for the DezTechYug website.

## Prerequisites
- Node.js 18.x or newer
- npm 9.x or newer

## Installation
Install the project dependencies:

```bash
npm install
```

## Running the development server
Start the development server on [http://localhost:3000](http://localhost:3000):

```bash
npm start
```

You can also use the explicit development command:

```bash
npm run dev
```

## Additional scripts
- `npm run build` – create an optimized production build
- `npm run serve` – serve the production build locally on port 3000 (run `npm run build` first)
- `npm run lint` – run lint checks
